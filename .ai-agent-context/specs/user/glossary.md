# Glossary

- **Purpose**: Glossary for Linguistic Alignment via **Ubiquitous Language**.
- **Constraint**: Minimize LLM misinterpretation of unique domain knowledge and internal jargon.
- **Role**: Mandatory reference for LLM communication and development participation.

## User Domain

- **User**: A person who uses the MyApo service to manage Apostille certificates. Identified primarily by email.
- **User Wallet (UserWallet)**: A self-custodial wallet associated with a User. It stores the XRPL address and public key.
- **Verifier**: A social login provider (e.g., Google, Kakao, Line, Naver) supported by Web3Auth.
- **Verifier ID (VerifierID)**: A unique identifier for a user provided by the Verifier.
- **XRPL Address**: A unique account address on the XRP Ledger.
- **Public Key**: The cryptographic public key associated with the User's XRPL address, used for server-side signature verification.
- **Nationality**: An ISO 3166-1 alpha-2 country code representing the user's nationality.
- **Apostille**: A form of authentication issued to documents for use in countries that participate in the Hague Convention of 1961.
