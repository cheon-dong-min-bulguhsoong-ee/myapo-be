# Diagrams 

확정된 기능의 Sequence Diagram 혹은 State Transition Flow에 대한 `.png`, `.jpg`, `.svg` 등의 이미지 파일을 보관합니다.

이미지 파일은 `.mmd` 확장자를 갖는 Mermaid Script를 대상으로 생성합니다.

```text
https://pypi.org/project/mink-cli/
```

- 해당 URL의 패키지를 다운로드해 터미널에서 사용하세요.


```bash
// install
pip install mink-cli

// usage
mink input.mmd -o output.png
```

- 위와 같이 사용합니다.

- Mermaid Script 파일의 확장자가 `.md`가 아닌 `.mmd`임에 주의하세요. `mink-cli`는 Mermaid 문법만 담긴 `.mmd` 파일을 입력으로 권장합니다.
